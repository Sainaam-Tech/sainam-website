# Sainam Technology — Official Website

This repository contains the source code for the official website of **Sainam Technology**, a website and MVP development company based in Maharashtra, India.

The website is built using **Hugo (Extended)** with the **Hugoplate** theme and deployed via **GitHub Pages**.

Live site: https://sainam.tech

---

## About Sainam Technology

Sainam Technology helps startups, educational institutions, and businesses design, build, and scale modern digital products.

Our primary services include:
- Website Development
- MVP Development for Startups
- Custom Web Applications
- SaaS Product Development

We serve clients across:
**Mumbai, Palghar, Vasai, Virar, and Dahanu**

---

## Tech Stack

- **Hugo (Extended)** — Static site generator
- **Hugoplate Theme** — TailwindCSS-based Hugo theme
- **Tailwind CSS** — Styling
- **Node.js** — Asset pipeline
- **GitHub Actions** — CI/CD
- **GitHub Pages** — Hosting
- **Microsoft Clarity** — User behavior analytics
